# Mallareddy College AI 🎓

RAG based AI chatbot that answers questions
about Mallareddy Engineering College accurately!

## Live Demo
🌐 **Try it here:** https://huggingface.co/spaces/Nithish2711/college-ai

## What it does
- Answers college queries accurately
- Uses RAG for context aware responses
- Deployed live on HuggingFace Spaces
- Anyone can use it from any device!

## Sample Questions
- "What is the attendance requirement?"
- "What is BTech CSE fee?"
- "What is highest placement package?"
- "Which companies recruit from college?"

## How it works
Your question
↓
Search relevant documents (ChromaDB)
↓
Send context + question to Llama3
↓
Accurate answer! ✅

## Tech Stack
- Python
- LangChain
- ChromaDB (vector database)
- HuggingFace Embeddings
- Groq API (Llama3)
- FastAPI
- Gradio

## Results
- 95%+ relevance accuracy
- Answers 5 college query types
- Live deployed on HuggingFace!

## How to run locally
pip install langchain-groq chromadb
pip install langchain-community gradio
python rag_basics.py
